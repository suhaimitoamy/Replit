# Prompt for Replit Agent

Build a web app called **Goldbro** inspired by Neurobro, but focused only on **gold / XAUUSD**.

## Goal
Create a clean web dashboard for gold traders and investors with these modules:
1. Daily Gold Brief
2. Ask Gold AI
3. Event Radar
4. Setup Scanner
5. Alert ideas

## UI Requirements
- dark premium dashboard
- responsive layout
- sidebar navigation
- top summary cards for XAUUSD, DXY, US10Y, and real yield
- cards for macro, technical, event, and AI modules

## Functional Requirements
- allow switching between different market scenarios
- show bullish, bearish, or neutral bias
- show support and resistance levels
- show long setup score and short setup score
- include a text box where users can ask questions about gold
- generate answers based on simple rule logic first

## MVP Data Model
Use mock data first for:
- gold price
- DXY
- US10Y yield
- real yield proxy
- event list
- daily brief
- setup score

## Pages / Sections
- dashboard
- brief
- event radar
- ask gold ai
- settings

## Tech Preference
Use a simple full-stack setup that can later connect to:
- live market APIs
- economic calendar API
- auth
- database
- notification system

## Future Expansion
Prepare the structure so it can later support:
- real-time alerts
- saved watchlists
- user history
- mobile-friendly notifications
